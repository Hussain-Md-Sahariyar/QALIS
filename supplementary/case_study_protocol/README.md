# Case Study Protocol

Research methodology documents for the QALIS empirical evaluation.
Four case study systems (S1–S4) were evaluated over a 3-month period (Oct–Dec 2024).

## Files

| File | Description |
|------|-------------|
| `case_study_protocol.md` | Full case study research protocol (Yin, 2014 framework) |
| `consent_form_template.md` | Participant consent form used for practitioner interviews |
| `data_management_plan.md` | Data storage, retention, access, and anonymisation policy |
| `irb_approval_summary.md` | IRB approval summary (ref: QUATIC-2025-IRB-Annex-B) |

## Study Overview

| Property | Value |
|----------|-------|
| **Study type** | Multiple-case study (Yin, 2014) |
| **Cases** | 4 production LLM-integrated systems (S1–S4) |
| **Study period** | 1 October – 31 December 2024 (92 days) |
| **Observations** | 3,400 per system (13,600 total) |
| **Interview participants** | 14 practitioners (Nov–Dec 2024) |
| **IRB reference** | QUATIC-2025-IRB-Annex-B |
