# Thematic Analysis

Framework-method thematic analysis of 14 practitioner interviews (Ritchie & Spencer, 1994).

## Files

| File | Description |
|------|-------------|
| `theme_summary.json` | High-level summary of all 8 themes |
| `coded_excerpts.csv` | All 237 coded excerpts with theme, code, interview, and IAA fields |
| `iaa_analysis.json` | Inter-rater reliability detail (Cohen's κ=0.81) |
| `TH1_theme_report.json` – `TH8_theme_report.json` | Per-theme evidence reports |

## Themes

| ID | Theme | n excerpts |
|----|-------|-----------|
| TH1 | Inadequacy of existing QA approaches | 38 |
| TH2 | Hallucination as primary safety concern | **52** |
| TH3 | Transparency as underserved quality gap | 48 |
| TH4 | Integration reliability distinct from model quality | 31 |
| TH5 | Need for operationalised thresholds | 27 |
| TH6 | Prompt brittleness as robustness failure mode | 19 |
| TH7 | Regulatory compliance as quality driver | 14 |
| TH8 | Tooling fragmentation as adoption barrier | 8 |
| **Total** | | **237** |

## Key Findings

- **TH3 (Transparency)** was cited unprompted by 11/14 participants — confirming
  TI as the most underserved QALIS dimension (mean score 7.05, σ=1.35).
- **TH2 (Hallucination)** produced the most excerpts (n=52), strongest in
  healthcare and legal sectors.
- **TH7 (Regulatory)** shows that compliance requirements drive quality investment
  above what teams would otherwise adopt — motivating domain-specific threshold overrides.
- **Inter-rater reliability**: Cohen's κ = 0.81 (substantial agreement).
